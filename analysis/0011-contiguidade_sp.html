Manipulação

Queen

Vizinhança por contiguidade:

nb_queen <- spdep::poly2nb(pl = shp_sp,
                          queen = TRUE,
                          row.names = shp_sp@data$NM_MUNICIP)

Visualização estrutural:

Visualização do critério Queen:

raster::plot(shp_sp, border = "lightgray",
         main = "Vizinhança por Contiguidade (Critério Queen)") + 
raster::plot(nb_queen, 
         coordinates(shp_sp), 
         add = TRUE, 
         col = "#33638DFF")

## integer(0)

Criação de matriz W com tratamento para a ilha:

matrizW_queen <- spdep::nb2mat(neighbours = nb_queen,
                               style = "B",
                               zero.policy = TRUE)
colnames(matrizW_queen) <- shp_sp@data$NM_MUNICIP
